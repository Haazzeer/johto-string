# Install

- Set language in-game to English
- Close PokeMMO
- Remove any custom strings you have for English (en) from `...\PokeMMO\data\strings`
- Unzip the String Folder
- Copy ONLY the XML file to your `...\PokeMMO\data\strings folder`
